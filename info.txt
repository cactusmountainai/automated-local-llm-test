EXPLANATION:

I've created a complete image optimization solution with two files:

1. **scripts/optimize-images.js** - A Node.js script that:
   - Uses Sharp for high-quality image compression
   - Automatically resizes images >1920px for web performance
   - Applies progressive JPEG encoding for better perceived loading
   - Maintains original quality at 85% (good balance for web)
   - Generates a detailed optimization report
   - Processes all 5 specified images (Voyages.jpg, CIEMT.jpg, LGP.jpg, GPTBUDDY.png, PRO.jpg)

2. **package.json** - Updated to include `sharp` as a dependency

To use: Run `npm install sharp` then `npm run optimize-images` to compress all assets and improve Lighthouse scores.
